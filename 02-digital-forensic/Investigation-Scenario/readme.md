# Skenario Investigasi Forensik

Digital forensik merupakan skill yang harus terus dilatih dengan konsisten karena seorang investigator memerlukan keahlian analisis yang tinggi ketika melakukan investigasi pada kasus nyata. Untuk itu saya berlatih serta membuat beberapa skenario yang menyerupai kasus nyata untuk menajamkan skill analisis dan melatih daya analisis forensik.
